## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Muthu-Raja-M/profile.git
```

Navigate into the folder:

```bash
cd profile
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
